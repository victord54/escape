## Installation
### Requirements
* Maven >= 3.4.1
* Java >= 17

### Development
#### Run
```bash
mvn clean javafx:run
```

#### Test
```bash
mvn clean test
```

### Production
#### Build
```bash
mvn clean package
```

#### Run
```bash
java -jar target/escape-X.Y-Z.jar
```
> Replace X.Y-Z with the version number, e.g. `1.0-SNAPSHOT`.
